The live functionality is using newly developed components of the browsers ! It might not be compatible with your browser, or your OS. You can check the compatibility table here. It is recommended for the streamer to use Chrome on Windows, otherwise audio might not be captured. Don't forget to also capture the audio when selecting the device/window to start the live from.
Due to the nature of the live functionality, there is an approximative 30 secondes delay between the capture and the actual broadcast. This should be improved in the future.
